Uploading Data

Uploading data to the website is simple, all you need to do is send a POST request with content-type application/json to /result.

Your JSON must respect all the required fields, if not, you will get back a 400 :)

Example with curl: curl -H "Content-Type: application/json" -X POST -d '{"name":"Anonymous", "gpu":"GPU DUMMY TEXT","cpu":"CPU DUMMY TEXT","log":"So this will be a detailed log.","score": 1}' http://localhost:5000/result

If the resource has been added successfully then the status code of 201 is returned, any other status code can be considered an error. In case of error the response JSON will contain the error message.

The name field can be used with the search functionality from the toolbar to search for results.

The location header of the response contains the location of the submitted result.
